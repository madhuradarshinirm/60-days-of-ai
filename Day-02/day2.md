
# Day 2 – Prompt Engineering

## Objective

Learn the difference between a lazy prompt and an engineered prompt, and understand how prompt quality affects AI-generated responses.

---

## Lazy Prompt

### Prompt

```text
Create an image explaining Prompt Engineering.
```

### Output

- Generated a simple visual explanation.
- Limited context resulted in a generic output.
- No detailed explanation or examples were included.

---

## Engineered Prompt

### Prompt

Used the detailed prompt template provided in the ABTalks 60 Days Claude Challenge.

### Output

- Explained Prompt Engineering in beginner-friendly language.
- Included the importance of Prompt Engineering.
- Compared weak and improved prompts.
- Listed practical benefits of writing better prompts.
- Generated a LinkedIn-ready educational image with a detailed layout.

---

## Comparison

| Lazy Prompt | Engineered Prompt |
|-------------|-------------------|
| Short and vague | Detailed and structured |
| Generic response | Beginner-friendly explanation |
| Minimal context | Rich context and instructions |
| Basic visual | Professional educational content |
| Limited information | Complete learning resource |

---

## Key Learnings

- Prompt Engineering is the process of writing clear and specific instructions for AI.
- Better prompts produce more accurate and useful responses.
- Adding context, structure, and examples significantly improves AI output.
- Prompt quality directly impacts the quality of AI-generated content.
