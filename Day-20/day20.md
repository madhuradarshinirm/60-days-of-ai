# Day 20 – Face Puzzle Game

## Objective

Built an AI-generated Face Puzzle Game using Claude.

## Features

- Webcam Photo Capture
- Drag & Drop Puzzle
- Mobile Touch Support
- 3x3, 4x4 and 5x5 Difficulty Levels
- Timer
- Move Counter
- Leaderboard
- Restart Puzzle
- Shuffle Puzzle
- Responsive Design

## Learnings

- AI can generate complete HTML applications.
- Learned camera integration using getUserMedia().
- Learned drag-and-drop puzzle logic.
- Learned responsive game development.
- Learned browser-based image processing.
